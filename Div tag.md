div tag is an html tag mainly use for adding layout to the webpage 

here is how it is done 

```html 
<!DOCTYPE html>
<html lang="en">
	<head>
		<title>Sample Web Page</title>
		<meta charset="UTF-8">
		<!-- here you can add your css -->
		<style></style>
	</head>

	<body>
		<div>
			<h1></h1>
			<p></P>
		</div>
	</body>
</html>
```
